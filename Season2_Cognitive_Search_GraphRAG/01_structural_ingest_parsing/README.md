# 📑 Phase 1: Structural Ingest & Parsing

## 📂 파일 구성
*   `01_theory.md`: 구조적 청킹의 핵심 이론 및 GIGO 방지 설계론
*   `02_structural_ingest_parsing.ipynb`: 파이썬 텍스트 파서 구현 실습 노트북
*   `data/`: 실습용 산업 매뉴얼 데이터 저장 폴더
