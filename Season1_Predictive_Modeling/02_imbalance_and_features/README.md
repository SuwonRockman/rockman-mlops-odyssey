# ⚖️ Phase 2: Imbalanced Data & Feature Engineering

본 단계에서는 극소수로 발생하는 장비 고장 불균형 데이터의 기하학적 분포 불균형을 극복하고, 시계열 노이즈 억제 및 추세 추적을 위한 다양한 롤링 통계 피처를 생성합니다.

---

## 📂 목차 (Contents)

- **`01_theory.md`**: SMOTE(Synthetic Minority Over-sampling Technique)의 기하학적 선형 보간 합성 원리 및 불균형 대안 평가지표(F1-score, Recall) 수학적 정리
- **`02_imbalance_and_features.ipynb`**: SMOTE 오버샘플링 적용 및 지수 가중 이동 평균(EMA) 등 도메인 피처 엔지니어링 구현 실습 노트북
