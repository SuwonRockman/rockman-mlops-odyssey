# 🤖 Phase 1: Agent Collaboration Architecture

LangGraph를 활용하여 반도체 CMP 공정 모델이 이상 징후를 예측했을 때, 자동으로 매뉴얼 검색(GraphRAG)을 수행하고 엔지니어에게 티켓을 발행하는 조건부 분기 멀티 에이전트 시스템의 뼈대를 설계합니다.

## 📂 파일 구성
*   `01_theory.md`: LangGraph 상태 정의 및 조건부 에이전트 라우팅 기하 원리
*   `02_agent_collaboration.ipynb`: LangGraph 오케스트레이션 구현 실습 노트북
